Running Dragonia:
To run Dragonia, make sure all extracted files are in the same directory.  Then click Dragonia_Graphics.exe to run.
Art Needed: 
	Pixel size for backgrounds is 896x504.
	Pixel size for animations or abilities are 200x220.
	Pixel size for enemies is 100x110 (for now)
	
	Any additional backgrounds are always welcome.
